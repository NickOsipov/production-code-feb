Project Description

Setup Instructions
1. Clone the repository:
    ```bash
    git clone git@github.com:NickOsipov/production-code-feb.git
    cd production-code-feb
    ```
2. Create a project:
    ```bash
    uv init
    echo "3.12.8" > .python-version
    uv venv
    uv add --requirements requirements.txt
    ```
